# Herbal Tea Benefits - 60 Second Animated Infographic

## 📺 Overview
A beautiful, animated HTML5 infographic video about herbal tea benefits and Islamic wisdom on health. The video is exactly 60 seconds long with smooth transitions between slides.

## 💾 How to Use

1. Open `index.html` in a web browser
2. The 60-second video starts automatically
3. Use SPACEBAR to pause/resume
4. Open DevTools (F12) for manual slide navigation

## 🎬 Exporting as Video

To export this as an MP4 video file:

### Option 1: Using ScreenFlow / OBS
1. Open the HTML in fullscreen
2. Record your screen for 60 seconds
3. Export as MP4

### Option 2: Using FFmpeg + Browser Automation
```bash
# Using puppeteer-extra (Node.js)
npm install puppeteer
# Create a script to capture the video
```

### Option 3: Using Playwright
```bash
npm install playwright
# Record the page as video
```

## 🔧 Customization

### Change timings
Edit the `slideTimings` array in `script.js`

### Add new tea varieties
1. Create new SVG background in `styles.css`
2. Add new slide HTML in `index.html`
3. Update `slideTimings` array

### Modify colors
Edit CSS variables in `styles.css`:
- Primary color: `#1a5f3d`
- Accent color: `#27ae60`
- Gold accent: `#f4d03f`
